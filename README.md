# LINA THE HEROINE
## About
Lina the Heroine is about a young girl  who has to go through obstacles in a various environments to save a baby lost in the woods.
## Gameplay Features
collect coins
avoid enemies
move on tiles and platforms
## Controls(AZERTY Keyboard)
movement: move left with A, move right with D, jump with space bar. OR move  left and right with the arrow keys, and jump with space bar.
##Levels
level 1 to 4
##Build with
Godot 4.5.1
## AI Usage
I used AI for debugging and fixing code errors, I wrote all the codes myself while following youtube tutorials.
